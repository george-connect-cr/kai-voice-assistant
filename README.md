# Connect Prompt Library (GitHub Pages)

## Cómo publicar en GitHub Pages
1. Crea un repositorio (por ejemplo `connect-prompts`).
2. Sube estos archivos al root del repo:
   - `index.html`
   - `styles.css`
   - `app.js`
   - `prompts.json`
3. En GitHub: **Settings → Pages → Build and deployment → Deploy from a branch**.
4. Selecciona `main` (o `master`) y `/root`.

## Contenido
- 150 prompts (ES/EN) en `prompts.json`
- UI: búsqueda, filtros, categorías, drawer, copy, favoritos, recientes, tema oscuro y idioma.

## Nota de marca
Los colores son una aproximación inspirada en `connect.pr`. Ajusta los valores en `:root` dentro de `styles.css` si quieres clavar el brand book.
